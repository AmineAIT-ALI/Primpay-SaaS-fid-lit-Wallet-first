1. Parcours CLIENT FINAL ce qu'il voit

Le client final (ex. Marc, qui mange une pizza). Il ne télécharge aucune app. Tout passe par son Wallet natif.

1

Entre dans le commerce

Voit un sticker QR sur le comptoir : "Ajoutez votre carte fidélité"

→
2

Scanne avec l'appareil photo

Pas d'app à télécharger. Le scan ouvre une page web.

→
3

Saisit prénom + téléphone

30 secondes max. Aucun mot de passe.

→
4

Ajoute la carte au Wallet

Apple Wallet (iOS) ou Google Wallet (Android). Bouton natif.

→
5

Présente la carte à chaque passage

Le staff scanne le QR du pass Wallet. Voit sa progression.

→
6

Reçoit sa récompense

Au seuil atteint, le staff active la reward. Cycle recommence.

Ce que le client NE voit jamais : le backend, les règles métier, les autres commerces où il a peut-être une carte Primpay, les serveurs, les outils internes.

2. Parcours COMMERÇANT ce qu'il voit

Le commerçant (ex. Ali, pizzeria) utilise une PWA web sur son téléphone/tablette/écran caisse. Pas d'app native.

1

Ouvre la PWA staff

Installée en 2 taps sur écran d'accueil. Login email + mdp.

→
2

Écran principal (5 boutons max)

  • Scanner
  • Rechercher
  • Derniers clients
  • +1
  • Reward dispo
→
3

Crédite un client en < 2s

3 chemins : scan QR / recherche / derniers clients. Feedback immédiat.

→
4

Voit le dashboard

Clients inscrits, passages 7/30j, rewards émises/consommées.

→
5

Config programme

Type (points/tampons), seuil, récompense, cooldown, limite jour.

→
6

Gère sa facturation

Abonnement Stripe, factures PDF, résiliation 1 clic.

Ce que le commerçant NE voit jamais : les clients des autres commerces Primpay, le code, les logs techniques, les outils internes.

4. BACKEND — le cerveau invisible

NestJS monolith modulaire. API REST. Tout passe par ici : règles, validation, sécurité, multi-tenant.

Module AUTH

NestJS / JWT / bcrypt
  • Login staff (email + mdp)
  • JWT access 15min + refresh 30j
  • Rôles : owner / manager / staff

Module CUSTOMERS

NestJS + PostgreSQL
  • Création customer (flow QR public)
  • Recherche prénom / téléphone
  • Identité globale (1 par téléphone)

Module MERCHANTS

NestJS + PostgreSQL
  • CRUD merchant + locations + staff
  • Config programme (points/tampons)
  • Isolation multi-tenant stricte

Module WALLET

Apple PassKit + Google Wallet API
  • Génération .pkpass (Apple)
  • Génération JWT (Google)
  • Signature QR fixe HMAC
  • Update dynamique des passes

Module LOYALTY ⭐

NestJS + Redis locks
  • Endpoint POST /scan (validation QR)
  • Endpoint POST /loyalty/credit
  • Endpoint POST /loyalty/redeem (lock)
  • Cooldown + limite jour (Redis)
  • Génération auto reward au seuil

Module ANALYTICS

NestJS + SQL agrégés
  • Clients actifs 7/30/90j
  • Rewards émises/consommées
  • Rétention basique

Module BILLING

Stripe SDK + webhooks
  • Checkout abonnement
  • Webhooks Stripe (paid, failed, canceled)
  • Gestion plans Solo/Multi
  • Gestion trial 14j

Module ADMIN (nous)

Accès restreint Vito
  • Vue globale tous merchants
  • Adjustments exceptionnels
  • Support : reset pass, désactivation

5. INFRASTRUCTURE — où tout tourne invisible

Hébergement physique + services tiers qui font tourner le cerveau.

VPS principal

OVH / Hetzner / Scaleway (EU)
  • 4-8 vCPU / 8-16 Go RAM au démarrage
  • Docker Compose (API + DB + Redis)
  • Nginx reverse proxy
  • Coût : ~20-40€/mois

Base PostgreSQL

Version 16+
  • Sur le VPS (MVP) puis managed
  • Backups quotidiens automatiques
  • Rétention 30 jours
  • Stockage off-site (S3/Backblaze)

Redis

Version 7+
  • Cooldown par client
  • Rate-limit API
  • Locks transactionnels (redeem)
  • Cache programmes

Cloudflare

DNS + Edge + WAF
  • DNS + SSL/TLS
  • Protection DDoS basique
  • Cache static landing
  • Gratuit au démarrage

Stripe

Paiements SaaS
  • Abonnements récurrents
  • Webhooks pour statuts
  • Factures générées auto
  • Frais : 1,5% + 0,25€ par transaction

Apple Developer + Google Wallet

Programmes officiels
  • Apple : 99$/an (certificat PassKit)
  • Google Wallet API : gratuit
  • Obligatoires pour émettre passes

Email transactionnel

Resend / Postmark
  • Bienvenue, invitations, relances
  • ~10-20€/mois au démarrage

Monitoring

Sentry + Better Stack
  • Erreurs backend (Sentry free tier)
  • Uptime 3 régions (Better Stack)
  • Alertes Discord/SMS
Recommandation VPS : Hostinger VPS est possible mais limite sur la stabilité pro. Préférer Hetzner Cloud (Allemagne) pour le rapport qualité/prix (~5-15€/mois pour un CX22, montée en charge facile) ou Scaleway DEV1-S (France, RGPD friendly ~8€/mois). OVH VPS reste une valeur sûre française.

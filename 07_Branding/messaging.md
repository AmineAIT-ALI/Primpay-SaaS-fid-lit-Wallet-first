# Messages clés

## 1. Objectif du document
Lister les messages à marteler en cohérence, sur tous les supports.

## 2. Message principal (master line)

> Plus simple qu'une app. Plus moderne qu'une carte papier.

À utiliser sur : landing page, bio Instagram, carte de visite, premier slide de pitch.

## 3. Messages secondaires

### Pour commerçants
- "Prêt en 10 minutes."
- "29 € / mois, sans engagement."
- "Vos clients n'ont pas besoin d'app."
- "Installation offerte si vous signez aujourd'hui."

### Pour clients finaux
- "Une carte dans votre Wallet. Plusieurs commerces. Zéro app."
- "Ajoutez votre carte en un scan."
- "Votre fidélité, directement dans votre téléphone."

### Pour prospects sceptiques
- "Essayez 14 jours, sans carte bancaire."
- "Résiliez en 1 clic."
- "Vos données, vous les gardez."

### Pour investisseurs / partenaires (Phase 2+)
- "L'infrastructure de fidélité pour le commerce physique."
- "Effet réseau dès la 10e signature locale."

## 4. Règles d'usage

### Cohérence
- Toujours les mêmes formulations, pas de variantes créatives improvisées.
- Les formulations évoluent volontairement, tracées dans ce document.

### Hiérarchie
- 1 message principal sur chaque support (affiche, landing, pitch).
- Maximum 2 messages secondaires si l'espace le permet.

### Tone of voice
- Phrases courtes.
- Pas de superlatif sans preuve.
- Pas de jargon SaaS ("empowering", "révolutionner", "disrupter").

## 5. Bannissements

| ❌ À éviter | ✅ Remplacer par |
|---|---|
| Révolutionnaire | Nouveau / moderne |
| Game-changer | Vraie différence |
| Disrupter | Remplacer |
| Empowering | Aider / équiper |
| Innovant | *(rien, montrer plutôt)* |
| Solution clé en main | Prêt en 10 minutes |
| Augmentez votre CA de X% | Faites revenir vos clients |

## 6. Messages par canal

### Landing page
- H1 : message principal.
- Sous-titre : 2 bénéfices concrets.
- Section démo : 1 GIF + 1 phrase.
- CTA : "Démarrer l'essai gratuit".

### Instagram
- Bio : "Carte de fidélité dans le Wallet. Pour les commerces de quartier."
- Posts : avant/après, démos 15 secondes.
- CTA stories : "DM pour démo gratuite".

### Démos terrain
- Script complet dans `script-demo-terrain.md`.

### Emails transactionnels
- Bienvenue : chaleureux, utile, court.
- Relance : une seule action claire.
- Fin d'essai : cadré mais pas anxiogène.

## 7. Messages à adapter par segment (cercle 1)

### Pizzeria
- "10 pizzas offrent la 11e. Automatisé."

### Coffee shop
- "Vos habitués du matin, fidélisés sans carte papier."

### Boulangerie
- "Un sourire au comptoir. Un tap sur le téléphone. C'est fait."

### Kebab / snack
- "Fidélité rapide pour un service rapide."

## 8. Décisions figées
- Master line figée : "Plus simple qu'une app. Plus moderne qu'une carte papier."
- Bannissement du jargon SaaS et des superlatifs non prouvés.
